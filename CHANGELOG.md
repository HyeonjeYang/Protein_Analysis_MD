# Changelog

## v0.1.0 - 2026-06-21

Initial public pre-alpha release of `protein_analysis_md`.

- Added CALVADOS-compatible workflow preparation without vendoring CALVADOS.
- Added sequence, PTM, multi-protein, and pre-cleavage design scaffolds.
- Added local/HPC execution helpers, tmux launch, progress watch, and finalize.
- Added trajectory analysis, WT-vs-perturbation comparison, figures, reports,
  PyMOL export, and local HTML dashboards.
- Added GPL-3.0-only licensing files, citation metadata, audit notes, and CI.

This release is intended for research workflow development and early testing,
not for unsupported scientific production claims.
